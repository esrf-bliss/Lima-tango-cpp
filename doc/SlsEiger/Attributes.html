gapPixelsActivation false false Scalar READ_WRITE Tango::DEV_BOOLEAN OPERATOR Set/get gap pixels management activation value.
edgePixelsCorrection false false Scalar READ_WRITE Tango::DEV_DOUBLE OPERATOR Set/get the coefficient used for the correction of edge pixels values.

Indeed, the physical pixels at the border of the chips in the sensor are double in size.

The coefficient value will be close to 2.0 but must be tweaked by taking into account

the threshold energy.

This attribute is used when the gapPixelsActivation is enabled.
cornerPixelsCorrection false false Scalar READ_WRITE Tango::DEV_DOUBLE OPERATOR Set/get the coefficient used for the correction of corner pixels values.

Indeed, the physical pixels in the corner between chips are four-times the normal size.

The coefficient value will be close to 4.0 but must be tweaked by taking into account

the threshold energy.

This attribute is used when the gapPixelsActivation is enabled.
